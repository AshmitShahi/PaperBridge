# **App Name**: PaperBridge

## Core Features:

- Semantic Search Interface: A user-friendly search bar and a card-based results display for discovering research papers based on topics.
- AI-Powered Paper Retrieval: Backend integration leveraging embeddings (Sentence Transformers) to retrieve semantically similar papers to the user's query.
- AI-Generated Summaries: Each paper displayed includes a concise, easy-to-understand summary generated by an LLM to simplify complex information.
- Advanced Filtering and Sorting: UI elements allowing users to filter results by publication year, domain, open-access status, and sort by relevance or citation count.
- Related Papers Recommendation Display: Visually presents recommendations for semantically similar papers to the currently viewed or searched results.
- Paper Access and Download: Buttons on each paper card provide direct links to view the paper online and, if available, download the PDF.
- Paper Bookmark and Storage: Enables users to save interesting papers to a personal collection, with metadata stored in the PostgreSQL database for retrieval.

## Style Guidelines:

- A light color scheme that emphasizes clarity and focus, drawing inspiration from the analytical nature of research.
- Primary color: A sophisticated and professional deep blue (`#336699`), conveying depth of knowledge and trustworthiness.
- Background color: A very light, almost off-white blue-grey (`#EBEFF3`), providing a clean canvas for content without distraction.
- Accent color: A vibrant sky blue/cyan (`#5DC4DB`), used sparingly for interactive elements, highlights, and calls to action to add a touch of discovery and freshness.
- Headline and body font: 'Inter', a grotesque sans-serif that offers a modern, neutral, and highly readable appearance, suitable for both titles and detailed abstracts, aligning with the clean and professional design goal.
- Utilize crisp, modern line-art icons that complement the clean UI, representing academic and search-related actions such as save, download, and filter.
- A clean, minimalistic card-based layout for search results, featuring a prominent, centrally located search bar akin to familiar search engines. Emphasis on good information hierarchy within each paper card.
- Subtle and smooth loading animations to indicate active processing during search queries and API calls, enhancing the user experience without being intrusive.
